5G NR Tools

These interactive analyzers show how good, normal, and degraded 5G NR environments differ across the stack. Use the presets and parameter sliders inside each tool to connect impairments at one layer with visible behavior in the others.

Layer 1SignalScopeLayer 2FrameScopeLayer 3NetScope

Layer 1

SignalScope

RF / Air Interface

Open full screen

Physical-layer view of spectrum shape, I/Q constellation, EVM, BLER, and active RF impairments.

Layer 2

FrameScope

Resource Grid and Protocol

Open full screen

MAC, RLC, and PDCP behavior through HARQ activity, resource-grid allocation, retransmissions, and scheduling efficiency.

Layer 3

NetScope

Traffic and Data

Open full screen

Network and application-layer quality through throughput, latency, jitter, loss, TCP behavior, capacity, and QoE.
