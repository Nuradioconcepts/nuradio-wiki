Cable & Antenna Analyzer

Use this virtual tool to test practical RF line considerations before field work: cable type and length, connector type/gender/condition, antenna match quality, intended frequency range, and likely fault scenarios. The dashboard computes return loss, VSWR, single-port loss, reflected power, cable loss, and estimated distance to fault.

Start here: use the green Start Here: Help button in the top-right corner of the tool. It opens a full guide for all tabs (Build, Measure, Diagnose, Learn) with workflow steps, tab-specific instructions, variable definitions, and pass/check thresholds.
