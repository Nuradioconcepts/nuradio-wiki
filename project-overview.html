Project Overview

What It Is​

nuradio atlas is a Docusaurus 3 static documentation site — a living knowledge base covering 5G NR, O-RAN, packet analysis, resource references, and interactive tools. It's built as a pure docs site (no blog, no landing page) where the docs root (/) serves directly as the homepage.

Tech Stack​

  • Framework: Docusaurus 3.9.2 with the @docusaurus/preset-classic preset
  • Language: JavaScript/JSX with TypeScript type-checking support
  • React: v19 (for custom components)
  • Node: >=20.0 required
  • Syntax highlighting: Prism React Renderer (bash, yaml, json enabled)

Content Model​

  • All pages are MDX files — standard Markdown with optional JSX component imports
  • Content is organized into top-level sections, each with an overview.mdx landing page linked from sidebar category headers
  • The current primary sections are: Dashboard, Project Overview, 5G NR (procedures-focused), O-RAN, Packet Analysis, Resources, Tools, and Glossary
  • The sidebar is manually defined in sidebars.js — new pages must be explicitly added there to appear in navigation

Key Configuration Decisions​

  • Docs-only mode: routeBasePath: '/' means docs are served at the root — there is no separate homepage route
  • Dark mode only: Color mode switching is disabled; the site is permanently dark
  • No edit links: editUrl is set to undefined — there's no "Edit this page" button
  • Auto-collapse sidebar categories: Enabled; only the active section stays open

Adding Content​

  1. Create a .mdx file in the appropriate docs/ subdirectory
  2. Add a frontmatter title (and optionally description) at the top
  3. Register the file path in sidebars.js under the correct category
  4. Import any custom components from src/components/ as needed

Local Development​

npm install        # install dependencies
npm start          # dev server at localhost:3000 (hot reload)
npm run build      # production build → /build
npm run serve      # preview the production build locally
