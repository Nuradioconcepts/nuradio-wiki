# 5G NR Channels

This is a list of observed 5G NR downlink channels captured from live network scans. Each entry represents a detected carrier and includes the ARFCN, GSCN, operating band, center frequency, bandwidth, subcarrier spacing (SCS), SSB parameters, and derived carrier edges. Click any column header to sort the table.
